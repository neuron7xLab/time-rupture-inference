# Code of Conduct

## Purpose

This project is a public research and engineering artifact. Discussion must preserve technical clarity, falsifiability, and respect for contributors.

## Expected behavior

Participants are expected to:

- discuss claims, evidence, and mechanisms directly
- provide reproducible examples when reporting defects
- separate scientific disagreement from personal attack
- respect project boundaries and maintainer time
- keep public reports free of private data or secrets
- use issue templates and pull request templates when contributing

## Unacceptable behavior

Unacceptable behavior includes:

- harassment or personal attacks
- spam or low-effort issue flooding
- posting private data or credentials
- deliberately misleading claims about project capability
- bypassing contribution and review process
- hostile behavior toward maintainers or contributors

## Enforcement

The maintainer may close, lock, edit, or remove issues, pull requests, or comments that violate this code.

Repeated violations may lead to blocking from repository interaction.

## Scope

This code applies to repository issues, pull requests, review comments, and other project-maintained spaces.

## Reporting

For conduct issues, open a repository issue using the closest matching template and keep the report factual.
