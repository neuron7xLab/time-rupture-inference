# RESIDUAL RISKS (open, stated — not hidden)

1. **No real external collaborator run.** The redacted private-layer
   path is demonstrated only with a built-in mock probe. Until an
   independent person runs it on a real private hypothesis, status is
   capped at CONDITIONALLY_READY. This is the single blocking fact.

2. **Co-designed degenerate probe.** The adversarial battery rejects 8
   probe classes and a family-sensitivity scan, but a probe
   specifically engineered against this exact scan set is not proven
   impossible. Coverage is improved, not closed.

3. **Synthetic only.** Every benchmark is a synthetic family. No
   real-world transfer is claimed or tested.

4. **Characterized open boundary.** A location-contrast statistic
   cannot separate a stationary bimodal mixture from a level shift
   (portfolio lineage #29, sealed PARTIAL). A distributional observable
   is the proposed — not run — next lineage.

5. **Empirical-quantile calibration.** The held-out-null threshold has
   no finite-sample distribution-free guarantee (#27 showed 2/12 fresh
   null draws can exceed). It is approximate by construction.

6. **Surface-claim drift.** External text is gated by claims-lint and a
   stricter surface-hardening test, but lexical gates cannot catch
   every rhetorical inflation; reviewer scrutiny is still required.

None of these are resolved by editing a threshold. Each is either a
decision-gated next experiment or an explicit scope statement.
