# CTI-OS v9 Neural Temporal Adapter Preregistration

## Allowed claim
A small neural temporal adapter is introduced and tested as a candidate agent substrate inside the existing deterministic verification system.

## Forbidden claim boundary
Do not claim intelligence, cognition, AGI, brain simulation, or biological time perception.

## Contract
- Input: observed scalar interval stream, optional previous prediction error.
- Causal-only processing: no future leakage.
- Output per step: prediction, error, uncertainty.
- Deterministic replay with pinned seed.
- Compare against scalar learned baseline.
- Negative result is valid.
